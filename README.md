# 2048 3x3

A 3x3 version of the classic 2048 game — a static page with no build step and no
framework. Soft rounded design, a combo streak, and automatic light/dark mode.
[Play Here](https://2048.club/)

## How to play

Use the arrow keys (or `WASD` / Vim `HJKL`), or swipe on touch devices, to slide
all tiles. Tiles with the same number merge into one. Chain merges across moves
to build a combo streak. Merge tiles to score high!

## Deploy

Pushing to `main` publishes to GitHub Pages via `.github/workflows/deploy.yml`.
